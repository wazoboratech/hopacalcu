A simple command-linecalculator built with python. Made for my programming assignment
##feactures
**basic operations**: Add "+", Subtract "-" , multiply "*", Divide "/"
**Error handling**: Cathes division by zero + invalind number inputs
**History**: 'h' to see last 5 calculations
**clean**: Type 'n' at "calculate again?" to quit

## how to run
1. make sure you have python 3 installed
2. open terminal/powershell in folder with 'h.py'
3. Run: 'python h.py'
